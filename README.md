# Калибровка

Калибровочная игра на критическое мышление. 15 утверждений, шкала-«прибор» во всю ширину:
тянешь стрелку вправо — «правда», влево — «миф», чем дальше от центра — тем выше заявленная
уверенность. Один жест = ответ + ставка. На обложке — интерактивная разминка с мгновенным
разбором (туториал), дальше 15 вопросов, а весь разбор — на финальном экране-сертификате.
Мобайл-фёрст, RU по умолчанию, есть переключатель на EN.

Никакого билда — чистые HTML/CSS/JS, работает прямо с GitHub Pages.

## Структура

- `index.html`, `style.css` — разметка и стили (аскетичный «прибор»: JetBrains Mono + PT Serif, бумага/чернила/латунь)
- `questions.js` — 15 вопросов + разминка `WARMUP` (RU/EN, категория, правильный ответ, объяснение)
- `i18n.js` — все тексты интерфейса на RU/EN
- `app.js` — вся игровая логика: шкала-ставка, скоринг (правило Бриера), архетипы, результат
- `config.js` — сюда вписывается URL веб-хука для сбора данных
- `apps-script/Code.gs` — код для Google Apps Script (принимает данные и пишет их в Google Sheet)

## Скоринг

Очки за вопрос считаются правилом Бриера (строго правильное правило оценки — честная
уверенность максимизирует ожидаемые очки). Для наглядности итог линейно масштабируется
в 0–100%: не двигал ползунок нигде → 50% («сигнала нет»), уверен и всегда прав → 100%,
уверен и всегда неправ → 0%.

## Сбор данных (Google Sheets)

Игра шлёт три типа событий: `start` (начало игры), `answer` (каждый ответ),
`summary` (итог сессии). Чтобы данные реально сохранялись:

1. Создай новую Google Таблицу (sheets.new).
2. `Расширения → Apps Script`.
3. Удали содержимое `Code.gs` в редакторе и вставь туда содержимое файла
   [`apps-script/Code.gs`](apps-script/Code.gs) из этого репозитория.
4. `Развернуть → Новое развёртывание` → тип **Веб-приложение**.
   - Выполнять от имени: **Меня**.
   - У кого есть доступ: **Все**.
5. Разреши доступ (появится предупреждение — это твой собственный скрипт, жми «Advanced» → «Go to... (unsafe)» → Allow).
6. Скопируй URL веб-приложения (заканчивается на `/exec`).
7. Вставь его в [`config.js`](config.js):
   ```js
   const CONFIG = {
     SHEETS_WEBAPP_URL: 'https://script.google.com/macros/s/XXXXX/exec',
   };
   ```
8. Закоммить и запушь — данные начнут появляться в таблице (листы `starts`, `answers`, `summary`
   создадутся автоматически при первом ответе).

Если `SHEETS_WEBAPP_URL` пустой — игра работает как обычно, просто ничего никуда не отправляет.

## Публикация

Сайт уже задеплоен через GitHub Pages из этого репозитория (ветка `main`, корень `/`).
Любой `git push` в `main` обновляет живую версию за 30–60 секунд.

## Локальный запуск

Любой статический сервер подойдёт, например:

```bash
npx serve .
```
